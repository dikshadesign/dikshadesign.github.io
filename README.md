
# Diksha Design Portfolio

Personal portfolio site for Diksha's industrial design work, sketches, and project case studies.

## Structure

- `index.html` - homepage
- `sketches.html` - sketches gallery page
- `projects/glide.html` - project detail template
- `assets/css/style.css` - shared site styles
- `assets/js/main.js` - shared site interactions
- `assets/images/favicon.svg` - favicon asset
- `resume/` - place resume PDF files here
